# Population Count & Bit Scan

Population count, leading/trailing zero and one counts, first/last set-bit index with valid, and power-of-two detection.

*Directory:* `src/popcount_bitscan/` -- 32 modules.

| Module | Description |
|--------|-------------|
| `first_one_index16` | 16-bit index of first (lowest) set bit. |
| `first_one_index32` | 32-bit index of first (lowest) set bit. |
| `first_one_index4` | 4-bit index of first (lowest) set bit. |
| `first_one_index8` | 8-bit index of first (lowest) set bit. |
| `is_power_of_two16` | 16-bit power-of-two detector. |
| `is_power_of_two32` | 32-bit power-of-two detector. |
| `is_power_of_two4` | 4-bit power-of-two detector. |
| `is_power_of_two8` | 8-bit power-of-two detector. |
| `last_one_index16` | 16-bit index of last (highest) set bit. |
| `last_one_index32` | 32-bit index of last (highest) set bit. |
| `last_one_index4` | 4-bit index of last (highest) set bit. |
| `last_one_index8` | 8-bit index of last (highest) set bit. |
| `leading_one_count16` | 16-bit leading-one count. |
| `leading_one_count32` | 32-bit leading-one count. |
| `leading_one_count4` | 4-bit leading-one count. |
| `leading_one_count8` | 8-bit leading-one count. |
| `leading_zero_count16` | 16-bit leading-zero count. |
| `leading_zero_count32` | 32-bit leading-zero count. |
| `leading_zero_count4` | 4-bit leading-zero count. |
| `leading_zero_count8` | 8-bit leading-zero count. |
| `popcount16` | 16-bit population count (number of 1s). |
| `popcount32` | 32-bit population count (number of 1s). |
| `popcount4` | 4-bit population count (number of 1s). |
| `popcount8` | 8-bit population count (number of 1s). |
| `trailing_one_count16` | 16-bit trailing-one count. |
| `trailing_one_count32` | 32-bit trailing-one count. |
| `trailing_one_count4` | 4-bit trailing-one count. |
| `trailing_one_count8` | 8-bit trailing-one count. |
| `trailing_zero_count16` | 16-bit trailing-zero count. |
| `trailing_zero_count32` | 32-bit trailing-zero count. |
| `trailing_zero_count4` | 4-bit trailing-zero count. |
| `trailing_zero_count8` | 8-bit trailing-zero count. |
